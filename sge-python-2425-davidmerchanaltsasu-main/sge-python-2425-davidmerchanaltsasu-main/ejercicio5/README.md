# Ejercicio5

Crea un programa que solicite al usuario un número del 1 al 10 y muestre por pantalla la tabla de
multiplicación del 1 al 10.
Ejemplo:
- 1 Introduce un número del 1 al 10: 3
- 3 x 1 = 3
- 3 x 2 = 6 
- 3 x 3 = 9 
- 3 x 4 = 12 
- 3 x 5 = 15 
- 3 x 6 = 18 
- 3 x 7 = 21 
- 3 x 8 = 24 
- 3 x 9 = 27 
- 3 x 10 = 30
